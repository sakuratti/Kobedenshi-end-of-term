|average|sunny|sunny magnification|rainy|rainy magnification|

|平均の入場者数|晴れの日|晴れの日の倍率|雨の日|雨の日の倍率|

- 行ごとにタイトル、公開日数(何週目か)の他要素は揃えている
- 平均的に席がよく埋まる上映は天候に左右されにくい
- 平日は天候が客数に影響する
- 休日は天候が客数に影響しにくい
- 休日は天候が埋まり方(2席毎に埋まりやすくなる)に影響する
